# Reality Explorer · 现实解析器

A computable, layered model of reality. Five layers (physical · biological · cognitive · social · digital), five interactive modules, one bilingual map.

## Modules

1. **Multi-Layer Reality Map** — 9 scales from quark (10⁻¹⁵ m) to global compute (10²⁵ flops), interactive zoom & layer detail
2. **Causality Engine** — pick a phenomenon (or type your own), see the cross-layer cause chain backward
3. **Hidden Structure Mode** — reveal the incentives / power / information flow beneath 6 familiar systems
4. **Reality Simulator** — 5 sliders (temperature, compute, inequality, fertility, attention); see ripple effects across all layers
5. **Knowledge Graph** — 34 concept nodes, ~60 edges, layer-filterable + searchable interactive SVG

## Features

- **Bilingual EN ↔ 中文** instant toggle, localStorage-persisted, no reload
- **5-color layer system** consistently applied across every module (cyan/green/violet/amber/magenta)
- **Live SVG visualization** for the reality map and knowledge graph
- **Procedural fallback** in the causality engine for arbitrary user input
- **Aggregate ripple computation** in the simulator across all moved variables

## Stack

Plain HTML + CSS + vanilla JS — single-file, zero build, Vercel static hosting.
